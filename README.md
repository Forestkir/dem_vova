## Что делаем?
Запускаем XAMPP, в нем включаем MySql и Apache (не забываем про порт 3306)
Запускаем PyCharm и создаем проект (можно любое название)
Создаем вместе с venv, если не работает, то нужно прописать в терминале .\.venv\Scripts\activate
Нужно докачать библиотеку pip install pyinstaller
далее открываем cmd и переходим на рабочий стол 
и пишем ссылку на гитхаб: gitclone https://github.com/Forestkir/dem_vova.git
не забываем установить все библиотеки, если в venv ничего нет!!
pip install PyQt5 mysql-connector-python pyinstaller Pillow
после гит клона переходим в ту папку, где у нас склонировался проект
на компах в ркси папка .git не будет отображаться, поэтому мы включаем отображение невидимых объектов и удаляем эту папку

ОЧЕНЬ ВАЖНО!

Мы не можем переносить данные в проект напрямую, нам нужно создать точно такую же структуру вручную, это же касается и csv файлов
И НЕ ЗАБЫТЬ УДАЛИТЬ 3 ФАЙЛА:
- папка dist
- папка build
- СтроительныеМатериалы.spec
потому что потом мы всё равно будем билдить проект заново
- все пнгшки и иконки перетащим из зипника, который нам даст комиссия

после того, как мы сделали структуру файлов (создали main, db, resources и т.д), то заходим в pycharm и пишем 
git status
должно написать: fatal: not a git repository (or any of the parent directories): .git

после этого мы переходим в phpmyadmin и там создаем базу (имя базы должно быть такое же, как в коде)

не забудь при создании бд указать unicode_ci, изначально стоит general_ci
далее переходим в import и там импортируем базу с помощью sql-скрипта, который у нас есть

вдруг если не получится зайти в проект, будет ошибка выскакивать, ТО НУЖНО ЗАЙТИ в XAMPP, открывать файлик myinit, найти строку password и поставить пробел после хештега (и перезапустить mysql), может опять ошибка выскочить, тогда:
нужно обратить внимание на порт 3307 или 3306

для того, чтобы собрать файл exe, мы прописываем данную команду (не забудь перейти в папку add)

pyinstaller --noconfirm --windowed --onefile --name "МирИгрушек" --icon "resources\icon.ico" --add-data "resources;resources" --collect-all mysql.connector app\main.py

чтобы залить коммит (локально), надо: 
(.venv) PS C:\Exam2026\app> cd .. 
(.venv) PS C:\Exam2026> git init Initialized empty Git repository in C:/Exam2026/.git/ 
(.venv) PS C:\Exam2026> git add . 
(.venv) PS C:\Exam2026> git commit -m "onecommit"
